## Sample 1 — Status update senior C-level (CEO/CBO)

```
Hôm nay đang migrate core mới 2 con cake cl & vds cl nên có thể impact, để em check thêm có thể do update database thôi ạ, số approve vẫn ổn
```
**Khi nào:** senior hỏi anomaly số → explain + reassure. Hedge với "có thể... thôi ạ".

```
hôm nay đang migrate vds nên số mới update tới 5hAM thôi nha anh
```
**Khi nào:** trả lời nhanh, "nha anh" peer-respectful.

```
Dạ anh em đang xem, data của BI đang đổi core nên số báo mỗi lúc một khác
```
**Khi nào:** report lý do số chưa stable. "Dạ anh em đang xem" = nhận trách nhiệm.

```
dạ pipeline data bị chậm thôi nha anh
```
**Khi nào:** trả lời ngắn — "Dạ" + lý do + "nha anh".

```
Legal mới phản hồi đầu tuần version đầu tiên, tụi em đang review à, rất nhiều comment vì lần đầu làm trên 100mil và quy trình semi-auto nên legal hỏi rất nhiều à
```
**Khi nào:** status report cho CEO. Hedge "lần đầu... nên... hỏi nhiều à". "à" trailing đặc trưng Đạm.

## Templates rút ra

| Mục đích | Template |
|---|---|
| Status report senior (hedge) | `Hôm nay [đang/vừa] [verb] nên có thể [impact], [explain hedge] ạ/nha anh` |
| Probe CRO/peer | `Anh/Chị [name + title] [xem/xin] [object] [open Q] ko` |
| Directive 1-on-1 | `[verb gốc] đi` (2-3 từ) hoặc `[topic] anh [verb] đi` |
| Loop-in request | `cho [person] aware luôn` |
| Insight với jargon | `[Subject] thì do [cause] nên [effect]` |
| Soft proposal | `nếu [condition] thì [action] không ta` |
| FYI senior pipeline | `Tụi em [verb past] với [partner], [next step] nha anh` |
| Hedge clarify | `[subject] thì chắc khó, + có khi [reason]` |
| Welcome cross-team | `Welcome [name + title] lên thuyền [domain] - [self-promote]` |

- Push back hard với senior không đồng ý

→ AI báo gap nếu gặp, không fabricate.

| Pattern | Sample |
|---|---|
| Strip scope ép action | "ko cần... cần biết lỗ nhiêu" · "P&l thui nha" · "quất thôi" |

| Concrete number + business frame | "20% của 8k tỷ" · "tăng 10% CVR" · "P&L ra balance nhiêu" |
| Informal "má"/"ông" với peer cùng cấp | "Đừng nói đi mwg nữa nha má" · "nhờ ông coi thử nha" |
