# Voice — Group "The Money Team"

> Samples khi Đạm chat trong group Money Team. Tone informal, action-oriented.
> *Last updated: 2026-05-13*

## Composition

- (cần feed) — group name implies team về money/finance/treasury
- Có vẻ là peer senior level — Đạm dùng "đi", "nha", "rồi" rất informal

## Markers

- Directive đầu hoặc cuối: "dô login đi", "nhanh cme đi"
- Update progress: "đang tạo project cho team", "anh/em cũng đang mò tiếp"
- Casual swear cho emphasis: "cme" (informal)
- "nha" trailing chỉ dẫn instruction

---

## Samples (Đạm)

```
nhanh cme đi
```
**Khi nào:** push action urgent. "cme" = informal emphasis (Cờ Mờ Nờ — bỏ chữ N), cấp peer thân.

```
team vào project chỗ actitvity thấy cái shared chat start here chưa nha
```
**Khi nào:** instruction kiểm tra adoption. Self-typo "actitvity" → real human.

```
mở bằng claude chỗ project file nha
```
**Khi nào:** hướng dẫn nơi access. Pattern: `[verb] bằng [tool] chỗ [location] nha`.

```
anh/em cũng đang mò tiếp để phối hợp cho hiệu qủa
```
**Khi nào:** update tình trạng đang figure out. "mò tiếp" = casual + exploratory.

```
đang tạo project cho team
```
**Khi nào:** progress update fragment.

```
sướng rồi
```
**Khi nào:** banter — happy with outcome.

```
dô login đi
```
**Khi nào:** push action. "dô" = "vào" miền Nam Việt.

---

## Patterns

| Mục đích | Template |
|---|---|
| Push urgent action | `nhanh [verb] đi` hoặc `dô [action] đi` |
| Progress update fragment | `đang [verb] cho team` |
| Instruction kiểm tra | `[subject] vào [location] thấy [thing] chưa nha` |
| Hướng dẫn truy cập | `mở bằng [tool] chỗ [location] nha` |
| Casual happiness | `sướng rồi` / `ok ngon rồi` |

## Anti-pattern cần cẩn thận

- "cme" chỉ dùng với peer thân, **không** dùng với C-level formal hoặc team mới
- "dô" miền Nam — match audience đã quen tone này
