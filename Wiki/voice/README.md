## Khi AI cần draft

1. Identify audience → pick file phù hợp
2. Đọc 2–3 samples gần với case → match pattern
3. Apply markers (pronoun, trailing, sentence length)
4. Draft → review với Đạm
5. Append message thật khi user gửi đi

## Edge cases CHƯA cover (báo gap)

- Email C-level Cake formal (CFO/CTO/Legal)
- Risk Committee talking points formal
- Push back hard với senior
- Bad news escalation (>P2)
- External partner email (formal English)

## Cadence update

- **Append** mỗi message hay
- **Distill pattern** 2 tuần/lần — update `_voice-pattern-notes.md`
- **Bump timestamp** khi đổi
